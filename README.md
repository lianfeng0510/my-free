# 校园贴吧

一个基于Flask和SQLite的简单校园贴吧网页原型。

## 项目结构

```
campus_board/
├── app.py              # 主应用文件
├── templates/
│   ├── base.html       # 基础模板
│   ├── login.html      # 登录页面
│   ├── register.html   # 注册页面
│   ├── home.html       # 首页（帖子列表）
│   ├── create_post.html # 发帖页面
│   └── post.html       # 帖子详情页面
└── README.md           # 运行指南
```

## 功能说明

1. **用户注册与登录**：支持用户名和密码注册，登录后可发帖和评论
2. **发帖功能**：登录用户可发布包含标题和内容的帖子
3. **帖子列表**：首页按发布时间倒序展示所有帖子
4. **帖子详情与评论**：点击帖子标题查看详情，可发表评论

## 技术栈

- **后端**：Python + Flask
- **数据库**：SQLite
- **前端**：HTML + CSS + JavaScript

## 运行步骤

### 1. 安装依赖

```bash
# 安装Flask及相关库
pip install flask flask-sqlalchemy flask-login
```

### 2. 运行应用

```bash
# 在项目目录下执行
python app.py
```

### 3. 访问网站

打开浏览器，访问 `http://127.0.0.1:5000/`

## 使用说明

1. **注册**：点击导航栏的"注册"链接，填写用户名和密码
2. **登录**：注册成功后，点击"登录"链接，使用注册的账号登录
3. **发帖**：登录后，点击"发帖"链接，填写标题和内容发布帖子
4. **查看帖子**：首页会显示所有帖子，点击标题进入详情页
5. **评论**：在帖子详情页下方填写评论内容并提交

## 注意事项

- 密码存储为明文，仅用于演示，实际项目中应使用哈希加密
- SQLite数据库文件会自动创建在项目根目录下
- 开发模式运行，不适合生产环境
