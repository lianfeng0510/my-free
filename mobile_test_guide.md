## 🔧 故障排除

### ❌ 无法访问
1. **检查网络**
   ```bash
   # 在电脑上检查IP
   ipconfig
   # 确保显示192.168.1.59
   ```

2. **检查防火墙**
   - 打开Windows Defender防火墙
   - 添加入站规则：允许端口5000
   - 或临时关闭防火墙测试

3. **检查服务器状态**
   ```bash
   # 查看端口占用
   netstat -an | findstr :5000
   # 应显示LISTENING状态
   ```

### ⚠️ 页面显示异常
1. **清除浏览器缓存**
   - 手机浏览器设置 → 清除缓存

2. **检查CSS/JS加载**
   - 打开开发者工具（F12）
   - 查看Console错误

3. **测试不同浏览器**
   - Chrome
   - Safari
   - 微信内置浏览器

### 设计规范
- **断点**: 480px (手机), 768px (平板)
- **触摸目标**: 最小44×44像素
- **字体大小**: 正文16px，标题按比例缩放
- **间距**: 使用rem单位，适配不同屏幕
